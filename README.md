# Проект: "ИИ-ассистент для фирмы Мастер на час"

## Описание проекта

Проект направлен на разработку интеллектуального ассистента, который будет автоматизировать взаимодействие с клиентами фирмы "Мастер на час". ИИ-ассистент сможет осмысленно вести диалог, отвечать на вопросы, основываясь на базе знаний, и управлять записью клиентов на встречи через интеграцию с Google Календарем. Это позволит оптимизировать рабочие процессы, повысить удовлетворенность клиентов и сократить время на рутинные задачи.

---

## Функционал

### 1. **Осмысленное ведение диалога**
   - Ассистент использует нейросеть для анализа входящих сообщений и генерации естественных ответов.
   - Способен поддерживать контекст диалога, чтобы обеспечить плавное и логичное общение.
   - Учитывает специфику запросов клиентов фирмы "Мастер на час".

### 2. **Ответы на вопросы по базе знаний**
   - База знаний содержит информацию о предоставляемых услугах, ценах, условиях работы и других важных аспектах.
   - Ассистент извлекает актуальные данные из базы знаний и предоставляет точные и информативные ответы.

### 3. **Запись клиента на встречу**
   - Ассистент может предложить клиенту доступные временные слоты для записи.
   - После согласования времени встреча автоматически добавляется в Google Календарь.
   - Отправляет клиенту подтверждение записи с указанием даты, времени и деталей встречи.

---

## Используемые сервисы

### 1. **Qwen**
   - Сервис используется для создания системного промпта, который задает правила поведения и структуру диалога для ИИ-ассистента.
   - Также Qwen помог в формировании базы знаний, которая служит источником информации для ответов на вопросы клиентов.

### 2. **Савви (Suvvy.ai)**
   - Платформа для создания и настройки ИИ-ассистента.
   - Обеспечивает интеграцию с внешними сервисами, такими как Google Календарь.
   - Предоставляет инструменты для тестирования и улучшения работы ассистента.

---

## Преимущества проекта

- **Автоматизация рутинных задач**: Ассистент берет на себя обработку запросов клиентов и управление расписанием, что позволяет сотрудникам сосредоточиться на выполнении основных обязанностей.
- **Повышение качества обслуживания**: Быстрые и точные ответы на вопросы клиентов способствуют улучшению их опыта взаимодействия с компанией.
- **Гибкость и масштабируемость**: Система легко адаптируется под изменяющиеся потребности бизнеса и может быть расширена для поддержки дополнительных функций.

---

## Этапы реализации

1. **Анализ требований**
   - Определение ключевых задач и целевой аудитории.
   - Сбор данных для формирования базы знаний.

2. **Разработка системного промпта**
   - Создание правил и шаблонов для работы ИИ-ассистента с использованием Qwen.

3. **Настройка базы знаний**
   - Структурирование информации о компании и услугах.
   - Интеграция базы знаний с ассистентом.

4. **Создание ИИ-ассистента**
   - Настройка диалогового интерфейса и интеграция с Google Календарем через Suvvy.ai.

5. **Тестирование**
   - Проверка работоспособности всех функций.
   - Корректировка промптов и базы знаний на основе обратной связи.

6. **Запуск и поддержка**
   - Внедрение ассистента в рабочий процесс.
   - Регулярное обновление базы знаний и мониторинг производительности.

## Перспективы развития

- Добавление функции анализа обратной связи от клиентов для дальнейшего улучшения сервиса.
- Интеграция с CRM-системой для более глубокой автоматизации бизнес-процессов.
- Расширение возможностей ассистента для работы с другими каналами коммуникации (например, телефонные звонки или чат-боты в социальных сетях).
